## 能先解决的无关文本，信息有用性（无用信息）

1.一些特殊的符号

2.引用的数字（问题很多,容易误判）  

（1）带括号,

```
[r'(\([\d\s,\.–]{1,20}\))',r'删除1:<u>\1</u>']
```

（2）不带括号，不带点特征

```
[r'([^\d])(\d{1,3}(\s?[–,]\s?\d{1,3}){1,4})([^\d])',r'\1删除2:<u>\2</u>'],
```

（3）不带括号，带点特征（加上点很容易误判）

```
[r'([\.,])(\s\d+([,\.]\d+){0,5})',r'\1删除3:<u>\2</u>'],
```

（4）只有一个数字（难清洗）

3.非正文内容，(参考文献，图片内的内容，图片的描述fig...等没有删除干净)

4.去除带有网址的句子,关键词   www、com、html、http

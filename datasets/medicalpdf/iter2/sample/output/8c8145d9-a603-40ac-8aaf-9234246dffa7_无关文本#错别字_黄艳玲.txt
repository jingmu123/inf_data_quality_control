## 6. 术后的语音训练

【1】(1) 语音康复锻炼: 嘱虑儿吹哨、吹气球、吹肥皂泡等训练,这样口腔内压力増高,腭咽闭合更须严密,软腭,尤以悬雍垂须提得更高。脣裂修复术后 1 个月开始锻炼,每日 6~12 次。 目的是要恢复正常的腭咽闭合,以便发出正常的语音。 
(2) 语音训练的内容: ①语音不良习惯的矫正训练: 膺裂患儿到了应该说话的年龄,由于发育器官的结构异常,语言条件反射不易建立。要矫正不良习惯。②腾咽闭合功能训练: 腭咽闭合功能的训练一般在术后 3~4 周开始,具体方法包括局部软腭按摩,以使癫痕软化; 作干呕、打呵欠和高声发 "啊" 音,以训练轶腭的拾高运动; 唇、舌和下领作多方运动,以训练唇、舌和下颌在语音活动中的协调性;口腔内鼓气以训练増加口腔内气压。③语音呼气节制训练 : 应通过吹蜡烛 . 吹气球乐器等方法进行训练。 ④ 语音技能发育训练 : 语音是一个复杂的功能系统,语音缺陷的个体差异其大。要教孩子正常的发音技能,语音技能的发育缺陷有关语音病理医师应有针对性地给以技能指导并矫正。⑤语音基本要素的训练:学发辅音时, 有些话音病理学家主张根据塞音, 鼻音, 边音, 蒙音和塞擦音的顺序, 先发塞音, 最后学塞擦音。最好按照正常婴儿开始发音的生理次序,即[ m ],[ b ],[ p ],[ w ],[ h ],[ n ],[ i ],[ d ], [ k ], [ g ], [ ng ], [ j ], [ ch ], [ f ], [ l ], [ r ], [ sh ], [ z ], [ s ]的次序进行训练。⑥单词和语句训练 : 在患儿掌握了拼音字母和单字拼音等语音基本要素之后 , 便可开始单词和语句的训练,并逐渐加长句子相加快速度。自己可练习唱歌,通诗,读报等。另外,还应给患者尽可能多地创造交谈的机会。 (沈立氏)黄艳玲

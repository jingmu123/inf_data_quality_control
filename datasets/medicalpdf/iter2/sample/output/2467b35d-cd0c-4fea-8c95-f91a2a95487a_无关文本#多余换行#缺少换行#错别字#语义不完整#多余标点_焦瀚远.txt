## 新型免疫检测技术

【1】应用近几年发展起来的灌注色谱的色谱介质固相化抗体、抗原或其他配位体充 !

【2】tectionsensors) o 传统的电化学免疫传感器是由具有识别和结合特性的抗体或抗原的分子识别元件与电化学电极组合而成。直接型电化学免疫传感器的测定原理是在免疫化学反应的同时，将反应的信息转变成电信号，间接型电化学免疫传感器则将免疫化学反应的信息转变为另一种中间信息，然后再将这中间信息转变成电信号。不论是直接型或间接型电化学免疫传感器所产生的电信号（电位或电流）通过电化学测量仪器进行检测。传统的电化学免疫传感器主要有五类，关于其分类、结构及测定原理，许春向教授等的专著曾作了较为详细的论述。 新型免疫检测传感器则由新型的免疫化学反应的识别元件构成的，且具有完全不同于传统检测法的检测系统和检测技术。

【4】免疫检测系统包括免疫检测传感器和高效液相色谱仪 ( 图 5 – 1 ) 。 免疫检测传感器: 由含有固相化抗体的贯穿孔珠 ( "Flow Through" Bead) 充填于柱体或筒体组成。主要的有 ID 传感 XL (ID sensor Xl), ID 传感器 CO (ID sensor CO)  及 ID 传感器 BA  (ID senor BA)。ID 传感器 XL 为抗体先结合然后交联在蛋白质 A 和蛋白质 C 表面，ID 传感器 CO 为共价配位体固相化作用，ID 传感器 BA 为生物素化的配位体 (biolinglated ligand) 固相化在抗生物素蛋白 (avidin) 表面。三种 ID 传感器的固相化作用示意图– 2 。

【5】## 5. 2

【6】免疫检测主要有三种方式: 结合/脱离免疫测定, 竞争性免疫测定和夹层式免疫测定。

【7】5.2. 1 结合/脱离免疫测定为最简单的免疫测定法。被测定的生物分子靶先结合于载体 上，选去未结合的物质后，抗原从载体上选脱以紫外分光检测器检出。检出限度为焦瀚远

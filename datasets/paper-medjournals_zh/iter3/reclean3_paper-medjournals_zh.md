## reclean3_paper-medjournals_zh

### 1.页码

例如：

于确诊SFT［2, 3，8, 9］。

中英文逗号同时在括号里，没有匹配到，增加英文逗号，补充到通用删除9：

```
[r'((\\)?[\[［][\d\s，,\-\–—]{1,}(\\)?[\]］])', r'通用删除9(中):<u>\1</u>'], 
```

### 2.无关链接补充

例如：（www.empowerstats.com，X&Y Solutions，Inc.Boston，MA）

补充到删除6：

```
[r'(\n|\(|（)((?:https?[：:][/／]{2}(?:www[.．])?|www[.．])[a-zA-Z0-9@／/-]+(?:[-.．／/][a-zA-Z0-9@／/_]{2,})+(?:[,，].*)?)(\n|$|\)|）)',r'删除6：<u>\1\2\3</u>'],
```

### 3.索引类文本补充

例如：

（批号：2020XE-GS083）

（摘自Carey WA, Weaver AL, Mara KC, et al. Inhaled nitric oxide in extremely premature neonates with respiratory distress syndrome[J]. Pediatrics, 2018, 141(3): e20173108. DOI：10.1542/peds.2017-3108.）

增加删除15：

```
[r'([(（](?:摘自|批号[:：]).*[）)])',r'删除15：<u>\1</u>'],
```

### 4.图片索引补充

例如：

图2A、B分别为大鼠呼吸气体采样装置的示意图与实物图。呼吸样本采集前，采用腹膜内注射水合氯醛水溶液（10%）的方法对大鼠进行麻醉，剂量为0.03 ml/kg。

见图3。
见图4。

增加删除15：

```
[r'(\n|^)(见?图[0-9A-Z]+.*)',r'\1删除15：<u>\2</u>'],
```
